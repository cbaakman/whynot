About

Why Not was developed at the CMBI by Tim te Beek (t.tebeek@cmbi.ru.nl). Feel free to contact me about any questions or requests concerning Why Not.

Technologies

A short list of the technologies used to build Why Not:
  • Java
  • Maven
  • Eclipse
  • Subversion
  • Spring
  • Hibernate
  • PostgreSQL
  • Wicket
  • Tomcat
  • JFreeChart
  • CXF
  • Log4J
  • JUnit
  • FindBugs
The sourcecode is freely available upon request and could easily be adapted to support other uses.
